设计元规范

所有原型必须遵守

硬约束

1. 全部颜色用 design-system.css 变量，禁用 hardcode hex
2. 正文字号 ≥ 16px；按钮高度 ≥ 44px；对比度 ≥ 4.5:1
3. 375px 宽度先做对，再放大到桌面
4. 禁用第三方 UI 库（纯 HTML+CSS+轻量 JS）
5. 图标用 inline SVG 或 emoji
6. 所有涉及用户数据的页面标注**加载 / 空 / 错 / 成功**四态
7. 合规文案严守："不诊断""不处方""仅供参考"醒目常驻
